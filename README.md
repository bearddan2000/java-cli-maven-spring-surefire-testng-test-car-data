# java-cli-maven-spring-surefire-testng-test-car-data

## Description
A POC for spring app using testng
framework with surefire plugin.

## Tech stack
- java
- maven
  - spring
  - testng
  - surefire

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
